# miniQuery

## Install
```
$ git clone https://github.com/oyvindhermansen/miniQuery.git && cd miniQuery
$ npm install
```

## Run
* For development use: `$ npm run develop`
* For production use: `$ npm run production`

## API

Available methods
```javascript
mQ(selector)
```
can chain these methods:

```javascript
html()
on()
loop()
addClass()
removeClass()
hasClass()
css()
wipeAttr()
addAttr()
getAttr()
destroy()
hide()
show()
```
